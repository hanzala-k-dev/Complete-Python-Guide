# 🐍 Complete Python & Data Science Guide

Welcome to the **Complete Python Guide**! This repository is a structured, step-by-step roadmap of my journey learning Python programming from the absolute basics to advanced data science libraries. It serves as both a personal portfolio and a comprehensive resource for anyone looking to master Python.

---

## 🗺️ Learning Roadmap

The repository is organized sequentially. Complete the foundational scripts first, then dive into the dedicated libraries.

### 1. Foundational Python (01 to 32)
These core files cover syntax, logic, control flow, and programming fundamentals:

*   **Syntax & Variables:** `01.Print_function.py` up to `05.Keyword_identifier.py`
*   **User Input & Operators:** `06.user_input_type_conversion.py` to `08.operator.py`
*   **Control Flow:** `09.indentation.py` to `15.break_continue_pass.py`
*   **Functions & Modules:** `16.built_in_function.py` and `17.built_in_modules.py` (and beyond!)

---

### 2. Data Science Libraries (33 to 36)
Once the basics are solid, the guide transitions into powerful libraries used for data analysis, manipulation, and visualization:

| Folder | Library | Description |
| :--- | :--- | :--- |
| 📂 **`33.NUMPY`** | **NumPy** | High-performance multidimensional arrays and mathematical operations. |
| 📂 **`34.PANDAS`** | **Pandas** | Data structures (DataFrames) and tools designed for structured data analysis. |
| 📂 **`35.Matplotlib`** | **Matplotlib** | The foundational library for creating static, interactive, and animated visualizations. |
| 📂 **`36.Seaborn`** | **Seaborn** | Advanced statistical data visualization built on top of Matplotlib. |

---
